3. РЕКОМЕНДУЕМЫЙ ПЕРВИЧНЫЙ ПОРЯДОК ДЕЙСТВИЙ

3.1 Установка приложения
Для установки приложения онлайн-школы программирования необходимо перейти на официальный сайт проекта: https://diploma.bogaledev.ru (Рисунок 1).
Рисунок 1

На главной странице сайта нужно выбрать версию приложения для используемой операционной системы: Windows / Linux. После этого начнётся скачивание архива с приложением. После завершения загрузки необходимо распаковать архив. Для операционной системы Windows необходимо распаковать архив приложения и запустить файл DiplomaApp.exe.
