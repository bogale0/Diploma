3. РЕКОМЕНДУЕМЫЙ ПЕРВИЧНЫЙ ПОРЯДОК ДЕЙСТВИЙ

3.1 Установка приложения
Установка приложения онлайн-школы программирования выполняется на основании релизной поставки в формате архива, содержащего каталог build.

Пользователь должен перейти на официальный сайт проекта: https://diploma.bogaledev.ru (Рисунок 1), выбрать версию для целевой операционной системы (Windows или Linux), загрузить архив и распаковать его в выбранный рабочий каталог.

Для операционной системы Windows после распаковки архива приложение готово к запуску. В релизе предварительно выполнено развёртывание необходимых Qt-компонентов (windeployqt), поэтому запуск осуществляется непосредственно через исполняемый файл DiplomaApp.exe.

Для операционной системы Linux после распаковки архива требуется установить недостающие системные зависимости и дополнительные пакеты Qt с использованием штатного пакетного менеджера дистрибутива. После установки зависимостей приложение запускается из распакованного каталога build.
