4. РАБОТА ПРОГРАММЫ

После запуска приложения пользователь видит главный экран платформы «Школа программирования» с навигационной структурой и переходом к списку доступных курсов. В верхней части окна расположена пользовательская панель управления, включая кнопку профиля и кнопку закрытия приложения. При первом входе в систему открывается раздел курсов (языков программирования), где можно выбрать интересующее направление для обучения.

Выбор курса выполняется нажатием на карточку соответствующего языка. После этого система открывает список тем, относящихся к выбранному курсу. Пользователь выбирает тему и переходит к странице теории, где размещен учебный материал по выбранному разделу. Теория отображается в текстовом формате и служит подготовительным этапом перед выполнением практической части.

На странице теории доступны ссылки/кнопки перехода к заданиям темы. При попытке открыть практическое задание система проверяет, авторизован ли пользователь. Если вход не выполнен, автоматически открывается окно авторизации. После успешного входа пользователь возвращается к целевому действию и продолжает работу без повторной навигации по разделам.

Экран авторизации поддерживает три режима:
- вход в существующий аккаунт;
- регистрация нового аккаунта;
- восстановление пароля.

При регистрации пользователь указывает логин, пароль, контрольное поле восстановления и роль (студент или преподаватель). После успешной регистрации система создает активную сессию и предоставляет доступ к защищенным разделам. В режиме восстановления необходимо ввести логин, данные для восстановления и новый пароль; после подтверждения можно сразу выполнить вход с обновленными учетными данными.

После авторизации студент получает доступ к:
- выполнению заданий;
- личному профилю;
- просмотру прогресса прохождения тем и курсов.

В профиле отображаются агрегированные показатели: общий процент прохождения, завершенные темы, незавершенные темы, а также прогресс по каждому курсу на основе количества освоенных тем. Эти данные используются для самоконтроля и планирования дальнейшего обучения.

Если пользователь авторизован как преподаватель, ему доступна панель преподавателя. В панели реализованы вкладки:
- «Курсы» — просмотр существующих курсов;
- «Новый курс» — добавление нового курса с описанием и ссылкой на изображение;
- «Новая тема» — добавление темы с теоретическим материалом;
- «Новое задание» — добавление практического задания для выбранной темы;
- «Новый тест» — создание тестовых проверок для задач.

При создании контента преподаватель заполняет поля формы и отправляет данные на сервер. После успешного сохранения информация становится доступной в общем каталоге и может быть использована студентами в рамках учебного процесса.

Для завершения работы пользователь может выйти из учетной записи через соответствующую кнопку. При выходе токен авторизации очищается, защищенные функции становятся недоступны до следующего входа, а навигация возвращается к базовым публичным разделам.
